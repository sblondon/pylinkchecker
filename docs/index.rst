Welcome to pylinkchecker's documentation!
=========================================


pylinkchecker is a standalone and pure python link checker and crawler that traverses a web site and reports errors (e.g., 500 and 404 errors) encountered.

pylinkchecker can also be used programmatically by calling one of the functions in pylinkchecker.api
